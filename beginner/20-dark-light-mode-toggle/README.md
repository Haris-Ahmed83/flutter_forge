# Dark Light Mode Toggle 🌓
> A beginner-friendly Flutter app demonstrating seamless theme switching.

## Description
This project showcases how to implement a dark and light mode toggle feature in a Flutter application. It's designed for beginners to understand core Flutter concepts related to theming and state management, providing a practical example of how users can personalize their app experience.

## Features
*   Effortless switching between dark and light themes.
*   Dynamic UI updates across the application upon theme change.
*   Utilizes Flutter's built-in `ThemeData` for comprehensive styling.
*   Implements `Provider` for efficient and scalable state management.
*   Simple and intuitive user interface for demonstration purposes.
*   Clean architecture suitable for beginners to learn from.

## Key Concepts Demonstrated
*   **ThemeData**: Used to define the visual properties of the Flutter application, including colors, typography, and widget themes, allowing for consistent styling across dark and light modes.
*   **Provider**: Employed for state management to efficiently notify widgets about theme changes, ensuring the entire UI updates without unnecessary rebuilds.

## Getting Started
To run this project locally, follow these steps:

1.  Clone the repository:
    ```bash
    git clone https://github.com/Haris-Ahmed83/dark_light_mode_toggle.git
    cd dark_light_mode_toggle
    ```
2.  Install dependencies:
    ```bash
    flutter pub get
    ```
3.  Run the application:
    ```bash
    flutter run
    ```
