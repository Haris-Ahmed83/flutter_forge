# Simple Calculator 🧮
> A basic, functional calculator app built with Flutter, demonstrating grid layouts and core logic.

## Description
This project implements a straightforward calculator application using Flutter. It provides a clean interface for performing basic arithmetic operations, making it an excellent example for beginners to understand UI layout and fundamental application logic in Flutter.

## Features
*   Perform addition, subtraction, multiplication, and division.
*   Clear button to reset the current input/calculation.
*   Support for decimal numbers.
*   Real-time display of current input and results.
*   Responsive button grid layout.
*   Basic error handling (e.g., preventing multiple decimal points).

## Key Concepts Demonstrated
*   **Grid layout**: Utilizing Flutter widgets like `GridView` or combinations of `Column` and `Row` to arrange buttons in a structured, responsive grid.
*   **Logic**: Implementing the core calculation engine, handling user input, managing application state, and processing arithmetic operations.


    ```bash
    flutter pub get
    ```

