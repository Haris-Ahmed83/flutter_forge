# 🧠 Quiz App
> Test your knowledge with an interactive quiz experience built with Flutter.

## Description
This is Project #7 from the `flutter_forge` series, a beginner-friendly Flutter application designed to create a simple yet engaging quiz experience. Users can answer a series of multiple-choice questions and receive immediate feedback, culminating in a final score.

## Features
*   Display a sequence of multiple-choice questions.
*   Allow users to select an answer for each question.
*   Provide immediate feedback on whether the selected answer is correct or incorrect.
*   Navigate seamlessly between questions.
*   Calculate and display a final score upon quiz completion.
*   Option to restart the quiz from the beginning.

## Key Concepts Demonstrated
*   **Conditional UI**: Dynamically updating the user interface based on application state, such as switching between question and results screens, changing button states, or displaying correct/incorrect feedback.
*   **Score**: Implementing a mechanism to track and calculate a user's performance throughout the quiz, typically by incrementing a counter for correct answers and presenting the final tally.

## Getting Started
To run this project locally, ensure you have Flutter installed, then follow these steps:

```bash
flutter pub get
flutter run
```
