# Random Quote Generator 💬
> Discover inspiration with a new random quote at your fingertips.

## Description
This Flutter project, part of the `flutter_forge` series, provides a straightforward application for generating random quotes. It's designed as a beginner-friendly introduction to handling data lists and implementing random selection in a mobile app, offering a simple way to get a new quote with each tap.

## Features
*   Displays a new random quote on demand.
*   Simple and intuitive user interface.
*   "Get New Quote" button for easy interaction.
*   Demonstrates basic UI updates using Flutter's state management.
*   Utilizes a predefined list of quotes for display.
*   Focuses on core Flutter widgets and logic.

## Key Concepts Demonstrated
*   **List**: How to define, store, and access a collection of data (quotes) within a Flutter application.
*   **Random**: Implementing logic to select an arbitrary item from a list, ensuring a different quote is displayed each time.

## Getting Started
To get this project up and running on your local machine, follow these steps:

1.  Clone the repository:
    ```bash
    git clone [repository-url]
    ```
2.  Navigate to the project directory:
    ```bash
    cd random_quote_generator
    ```
3.  Install the dependencies:
    ```bash
    flutter pub get
    ```
4.  Run the application:
    ```bash
    flutter run
    ```
