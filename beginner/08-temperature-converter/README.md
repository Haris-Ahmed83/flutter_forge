# Temperature Converter 🌡️
> A simple and intuitive Flutter application for converting temperatures between Celsius, Fahrenheit, and Kelvin.

## Description
This Flutter project, part of the beginner-friendly flutter_forge series, provides a straightforward temperature conversion tool. Users can easily input a temperature value and select their desired units to see instant conversions across common temperature scales. It's an excellent project for understanding basic UI input and logical processing.

## Features
*   Convert temperatures between Celsius, Fahrenheit, and Kelvin.
*   User-friendly input field for numerical temperature values.
*   Clear selection mechanisms for input and output temperature units.
*   Instantaneous display of converted results upon input or unit change.
*   A clean and responsive user interface for a smooth experience.

## Key Concepts Demonstrated
*   **Input:** Handling user input efficiently using Flutter widgets like `TextFormField` to capture numerical data. This includes input validation to ensure only valid numbers are processed.
*   **Conversion Logic:** Implementing the core mathematical formulas and conditional statements required to accurately convert temperature values between different scales (e.g., Celsius to Fahrenheit, Fahrenheit to Kelvin, Kelvin to Celsius).

## Getting Started
To get started with this project, ensure you have Flutter installed, then:

1.  Navigate to the project directory.
2.  Run the following commands in your terminal:

    ```bash
    flutter pub get
    flutter run
    ```
