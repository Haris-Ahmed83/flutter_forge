# 🔄 Unit Converter
> A simple and intuitive Flutter application for converting between various units.

## Description
The Unit Converter is a beginner-friendly Flutter project aimed at providing a practical tool for converting values across various units of measurement. This application serves as an excellent starting point for new Flutter developers to grasp fundamental UI elements, state management, and user interaction within a real-world context.

## Features
*   Effortlessly convert values between different units of measurement.
*   Intuitive user interface with clear input and output fields.
*   Dynamic unit selection using `DropdownButton` widgets.
*   Instantaneous conversion results as you type.
*   Support for multiple unit categories (e.g., length, weight, temperature).
*   Responsive design for various screen sizes.

## Key Concepts Demonstrated
*   `DropdownButton`: Used for selecting the "from" and "to" units in a user-friendly manner.
*   `TextFormField`: For capturing user input values that need to be converted.
*   State Management: Utilizing `setState` to update the UI based on user interactions and conversion logic.
*   Basic UI Layout: Employing widgets like `Column`, `Row`, and `Padding` for a structured interface.
*   Event Handling: Responding to changes in input fields and dropdown selections.

## Getting Started
To get a copy of the project up and running on your local machine for development and testing purposes, follow these simple steps.

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Haris-Ahmed83/flutter_forge.git
    cd flutter_forge/unit_converter # Or navigate to Project #17 folder
    ```
2.  **Install dependencies:**
    ```bash
    flutter pub get
    ```
3.  **Run the app:**
    ```bash
    flutter run
    ```
