# Color Picker 🎨
> A simple Flutter application to demonstrate interactive color selection and display.

## Description
This beginner-friendly Flutter project showcases how to create a basic color picker application. It focuses on using fundamental Flutter widgets to allow users to interactively change and display colors, providing a practical introduction to UI interaction.

## Features
*   Interactive color selection through user gestures.
*   Dynamically updates the displayed color based on user input.
*   Clean and intuitive user interface.
*   Demonstrates real-time UI updates in response to user actions.
*   Provides a foundation for more complex color manipulation tools.

## Key Concepts Demonstrated
*   `GestureDetector`: Used to detect various user gestures (e.g., taps, drags) and respond by changing the displayed color.
*   `Color`: Explores the `Color` class for representing and manipulating colors within Flutter, including creating and applying different color values.

## Getting Started
To get a copy of the project up and running on your local machine for development and testing, follow these simple steps.

1.  Clone the repository:
    ```bash
    git clone https://github.com/Haris-Ahmed83/flutter_forge.git
    cd flutter_forge/project_06_color_picker
    ```
2.  Install dependencies:
    ```bash
    flutter pub get
    ```
3.  Run the application:
    ```bash
    flutter run
    ```
