# Number Guessing Game 🎲
> Test your intuition and logical thinking in this classic number guessing challenge!

## Description
This Flutter project implements a simple yet engaging Number Guessing Game. Players try to guess a secret number within a predefined range, receiving hints after each attempt. It's a perfect project for beginners to grasp fundamental Flutter UI and basic game logic.

## Features
*   Generates a random secret number within a configurable range.
*   Provides an intuitive input field for user guesses.
*   Gives immediate feedback: "Too high", "Too low", or "Correct!".
*   Displays the remaining number of attempts.
*   Offers an option to restart the game with a new secret number.
*   Tracks and displays the number of guesses taken.
*   Clearly indicates game win and loss states.

## Key Concepts Demonstrated
*   **Random**: Utilizes `dart:math` to generate unpredictable secret numbers, forming the core challenge of the game.
*   **Logic**: Implements the game's rules, including comparing user guesses to the secret number, providing appropriate hints, managing the game state (e.g., attempts left, win/loss conditions), and handling user interactions.

## Getting Started
To run this project locally, ensure you have Flutter installed and then follow these steps:

1.  Navigate to the project directory:
    ```bash
    cd your-project-directory # e.g., cd flutter_forge/number_guessing_game
    ```
2.  Install the dependencies:
    ```bash
    flutter pub get
    ```
3.  Run the application:
    ```bash
    flutter run
    ```
