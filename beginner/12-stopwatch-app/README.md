# Stopwatch App ⏱️
> A precise and intuitive digital stopwatch built with Flutter, perfect for tracking elapsed time.

## Description
Project #12 in the beginner category, this Flutter application implements a fully functional stopwatch. It provides users with a clean interface to accurately track elapsed time, demonstrating core Flutter concepts for real-time data display and state management.

## Features
*   Start, pause, and reset stopwatch functionality.
*   Display elapsed time in hours, minutes, seconds, and milliseconds.
*   Record and display multiple lap times.
*   Clear and intuitive user interface for ease of use.
*   Real-time updates for a smooth and responsive time tracking experience.
*   Toggle between displaying current time and lap times.

## Key Concepts Demonstrated
*   **Timer**: Utilized for scheduling periodic tasks to update the stopwatch's elapsed time at regular intervals.
*   **StreamBuilder**: Leveraged to efficiently rebuild UI components whenever the stopwatch's time stream emits new data, ensuring real-time display updates without unnecessary widget rebuilds.

## Getting Started
To get a local copy up and running, follow these simple steps.

1.  Clone the repository:
    ```bash
    git clone https://github.com/Haris-Ahmed83/flutter_forge.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd flutter_forge/projects/stopwatch_app
    ```
3.  Install dependencies:
    ```bash
    flutter pub get
    ```
4.  Run the application:
    ```bash
    flutter run
    ```
