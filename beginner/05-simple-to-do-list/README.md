# Simple To-Do List 📝
> A minimalist Flutter application to manage your daily tasks with ease.

## Description
This project is a beginner-friendly Flutter application designed to help users keep track of their tasks. It focuses on demonstrating fundamental UI concepts and state management within a practical context, allowing users to add and remove to-do items dynamically.

## Features
*   Add new to-do items to a list.
*   Mark to-do items as completed (toggle completion status).
*   Remove existing to-do items from the list.
*   Clear all completed tasks with a single action.
*   Persist tasks locally (optional, but good for future enhancement ideas).
*   Responsive UI that adapts to different screen sizes.

## Key Concepts Demonstrated
*   **ListView**: Efficiently displaying a scrollable list of widgets, perfect for a dynamic list of to-do items.
*   **Dynamic State**: Managing and updating the application's UI based on user interactions, such as adding, deleting, or marking tasks as complete, using `StatefulWidget` and `setState()`.

## Getting Started
To get a local copy up and running, follow these simple steps.

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Haris-Ahmed83/flutter_forge.git
    cd flutter_forge/project_5_simple_todo_list
    ```
2.  **Install dependencies:**
    ```bash
    flutter pub get
    ```
3.  **Run the app:**
    ```bash
    flutter run
    ```
