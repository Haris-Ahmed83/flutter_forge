# Tip Calculator 💰
> Easily calculate tips and split bills with a clean, intuitive interface.

## Description
This Flutter project provides a simple yet effective tip calculator application. Users can effortlessly determine the appropriate tip amount for their service and even split the total bill among multiple people, all through a user-friendly interface. It's an ideal tool for quickly managing restaurant bills or other service charges.

## Features
*   Calculate tip amount based on a user-selected percentage.
*   Adjust tip percentage interactively using a smooth slider.
*   Input the total initial bill amount.
*   Dynamically display the total bill including tip.
*   Split the final bill (including tip) among a specified number of people.
*   Instant updates to calculations as inputs change.
*   Clear and intuitive user interface designed for ease of use.

## Key Concepts Demonstrated
*   **Slider:** Implementation of the `Slider` widget to allow users to select a tip percentage interactively and visually.
*   **math:** Utilization of basic arithmetic operations (addition, multiplication, division) to accurately calculate tip amounts, total bill, and per-person costs.

## Getting Started
To get a local copy up and running, follow these simple steps.

1.  Clone the repository:
    ```bash
    git clone https://github.com/Haris-Ahmed83/flutter_forge.git
    cd flutter_forge/tip_calculator
    ```
2.  Install Flutter dependencies:
    ```bash
    flutter pub get
    ```
3.  Run the application:
    ```bash
    flutter run
    ```
