# Age Calculator 🎂
> Calculate your age with precision and ease.

## Description
This Flutter application allows users to quickly determine their age based on a selected birth date. It provides an intuitive interface to pick a date and instantly displays the calculated age in years. A perfect beginner project to understand date handling in Flutter.

## Features
*   Intuitive user interface for date selection.
*   Accurate age calculation based on the selected birth date.
*   Displays age in years.
*   Utilizes Flutter's built-in `DatePicker` widget for an enhanced user experience.
*   Robust handling of `DateTime` objects for reliable calculations.
*   Simple and clean UI design.
*   Error handling for future dates (if applicable).

## Key Concepts Demonstrated
*   `DatePicker`: Demonstrates the use of Flutter's `showDatePicker` function to allow users to easily select a date from a calendar interface.
*   `DateTime`: Illustrates the creation, manipulation, and comparison of `DateTime` objects to perform accurate age calculations (e.g., finding the difference between two dates).

## Getting Started
To get a local copy up and running, follow these simple steps.

1.  Clone the repository:
    ```bash
    git clone https://github.com/Haris-Ahmed83/flutter_forge.git
    cd flutter_forge/projects/age_calculator
    ```
2.  Install dependencies:
    ```bash
    flutter pub get
    ```
3.  Run the application:
    ```bash
    flutter run
    ```
