🧠 Flashcard App
> Master new concepts with an engaging and interactive flashcard experience built with Flutter.

## Description
This is Project #16 from the `flutter_forge` series, a beginner-friendly Flutter application designed to help users learn and memorize information using digital flashcards. It demonstrates core Flutter concepts by providing a clean interface for flipping cards and navigating through a deck, making learning both simple and interactive.

## Features
*   Display flashcards with distinct front and back sides.
*   Smooth card-flip animation to reveal answers.
*   Intuitive navigation to move between flashcards.
*   Clear and simple user interface for focused learning.
*   Responsive design suitable for various screen sizes.
*   Easily extensible to add more flashcards or categories.

## Key Concepts Demonstrated
*   **Card flip**: Implemented using Flutter widgets that manage state to switch between the front and back views of a flashcard, often involving conditional rendering or a `Stack` widget.
*   **Animation**: Utilizes Flutter's built-in animation framework (e.g., `AnimatedBuilder`, `TweenAnimationBuilder`, or `Transform` widgets with `AnimationController`) to create a smooth and visually appealing rotational effect when a card is flipped.

## Getting Started
To get a local copy of this project up and running, follow these simple steps.

1.  Clone the `flutter_forge` repository:
    ```bash
    git clone https://github.com/Haris-Ahmed83/flutter_forge.git
    ```
2.  Navigate to the `project_16_flashcard_app` directory:
    ```bash
    cd flutter_forge/project_16_flashcard_app
    ```
3.  Install Flutter dependencies:
    ```bash
    flutter pub get
    ```
4.  Run the app:
    ```bash
    flutter run
    ```
