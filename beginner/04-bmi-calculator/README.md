# BMI Calculator 🏋️
> A simple and intuitive Body Mass Index calculator built with Flutter.

## Description
This project is a beginner-friendly BMI (Body Mass Index) calculator developed using Flutter. It allows users to input their height and weight, and then calculates and displays their BMI, along with a classification of their weight status. This application serves as an excellent starting point for understanding basic UI input and logical processing in Flutter.

## Features
*   Clean and user-friendly interface for easy input.
*   Calculates BMI based on user-provided height and weight.
*   Displays the calculated BMI value.
*   Categorizes BMI into standard classifications (e.g., Underweight, Normal, Overweight, Obese).
*   Input fields for height and weight with appropriate validation (e.g., numeric input).
*   Clear button to reset inputs and calculation.

## Key Concepts Demonstrated
*   `TextFields`: Handling user input for height and weight.
*   `Logic`: Implementing the BMI calculation formula and conditional logic for classification.

## Getting Started
To get a local copy up and running, follow these simple steps.

1.  Clone the repository:
    ```bash
    git clone https://github.com/Haris-Ahmed83/flutter_forge.git
    ```
    Navigate to the `bmi_calculator_project` directory within the cloned repository.
2.  Install Flutter packages:
    ```bash
    flutter pub get
    ```
3.  Run the app:
    ```bash
    flutter run
    ```
