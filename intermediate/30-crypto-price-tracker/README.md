# Crypto Price Tracker 📈
> Real-time cryptocurrency price monitoring at your fingertips.

## Description
"Crypto Price Tracker" is an intermediate-level Flutter project designed to display the live prices of various cryptocurrencies. It offers users an intuitive interface to keep track of their favorite digital assets, providing up-to-the-minute market data. This application serves as an excellent example of integrating real-time data streams into a Flutter app.

## Features
*   Display live prices for a curated list of cryptocurrencies.
*   Real-time data updates for dynamic price tracking.
*   Search and filter cryptocurrencies by name or symbol.
*   View detailed information for selected coins (e.g., 24h change, market cap).
*   Clean, user-friendly, and responsive UI.
*   Cross-platform compatibility (iOS, Android).
*   Error handling for network issues and API failures.

## Key Concepts Demonstrated
*   **WebSocket:** For establishing a persistent, full-duplex communication channel to receive real-time cryptocurrency price updates without constant polling.
*   **RESTful API Integration:** For fetching initial cryptocurrency listings, historical data, or additional coin details that might not require real-time streaming.
*   **State Management:** Implementing an efficient way to manage and update the UI based on incoming data streams and API responses.
*   **Asynchronous Programming:** Handling network requests and data streams effectively using `async`/`await` and Dart's Stream API.

## Getting Started
To run this project locally, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Haris-Ahmed83/flutter_forge.git
    cd flutter_forge/projects/crypto_price_tracker # Navigate to the project directory
    ```

2.  **Install dependencies:**
    ```bash
    flutter pub get
    ```

3.  **Run the application:**
    ```bash
    flutter run
    ```
