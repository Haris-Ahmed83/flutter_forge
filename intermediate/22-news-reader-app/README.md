# News Reader App 📰
> Stay updated with the latest headlines and articles.

## Description
This Flutter project, "News Reader App," provides users with a clean and intuitive interface to browse news articles from various sources. It demonstrates fundamental Flutter development practices by fetching and displaying dynamic data, making it an excellent example for intermediate-level learners keen on building data-driven applications.

## Features
*   Display a scrollable list of trending news articles.
*   View article headlines, sources, and short descriptions.
*   Tap on an article to view its full content (e.g., in a WebView or detail screen).
*   Implement pull-to-refresh functionality to fetch the latest news.
*   Handle network requests and display loading/error states.
*   Clean and responsive user interface design.

## Key Concepts Demonstrated
*   **JSON Parsing:** Efficiently parsing JSON data received from a news API into Dart objects to structure and display article information within the app. This involves using `dart:convert` and creating model classes.
*   **ListView:** Utilizing Flutter's `ListView` widget to dynamically render a scrollable list of news articles, optimizing performance for potentially large datasets through widgets like `ListView.builder`.

## Getting Started
To get a local copy up and running, follow these simple steps.

### Prerequisites
*   Flutter SDK installed (version 3.x.x or higher recommended)
*   Dart SDK
*   A code editor like VS Code or Android Studio

### Installation
1.  Clone the `flutter_forge` repository:
    ```bash
    git clone https://github.com/Haris-Ahmed83/flutter_forge.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd flutter_forge/project_22_news_reader_app # Adjust the folder name if different
    ```
3.  Install dependencies:
    ```bash
    flutter pub get
    ```
4.  Run the app:
    ```bash
    flutter run
    ```
