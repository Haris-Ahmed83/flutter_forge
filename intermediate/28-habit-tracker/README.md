# Habit Tracker ✅
> Track your daily habits, build better routines, and achieve your goals.

## Description
This Flutter project, "Habit Tracker," is designed to help users establish and maintain positive habits. It provides a straightforward interface to add, track, and manage daily routines, making it easier to stay consistent and monitor progress over time.

## Features
*   Add new habits with custom names.
*   Mark habits as completed or incomplete for any given day.
*   View habits and their completion status on a daily basis.
*   Navigate through a calendar to see past and future habit schedules.
*   Persistently store habit data locally on the device.
*   Simple and intuitive user interface for easy habit management.

## Key Concepts Demonstrated
*   **SharedPreferences**: Used for local data persistence, allowing the application to save habit information and user-specific tracking data across app sessions.
*   **Calendar**: Implementation of a calendar interface to display habits for specific dates and enable users to navigate through different days to track their progress.

## Getting Started
To get a local copy up and running, follow these simple steps.

1.  Clone the repository:
    ```bash
    git clone https://github.com/Haris-Ahmed83/habit_tracker.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd habit_tracker
    ```
3.  Install Flutter dependencies:
    ```bash
    flutter pub get
    ```
4.  Run the application:
    ```bash
    flutter run
    ```
